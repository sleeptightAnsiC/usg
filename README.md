### usg
Tiny program for generating Ulam Spiral and writing it as image file.

### BUILDING
Either:
* with GNU/make on POSIX (Linux, MacOSX, MinGW, etc):
```sh
$ git clone https://github.com/sleeptightAnsiC/usg.git
$ cd usg
$ make
```
* with any C99-capable compiler:
```sh
$ cc src/*.c
```
